# Projektuppgift, DT211G - Webbapplikation Mashup
Detta är ett projektarbete i _Webbutveckingsprogrammet_ på Mittuniveristetet.

## Länk till webbplats
[Min webbapplikation](https://dt211gmashup.netlify.app/)

## Syfte
Projektuppgiftens syfte är att omsätta kunskaper från tidigare moment och laborationer i kursen för att skapa en webbapplikation som använder minst två stycken APIer med hjälp av JavaScript, inga frontend-ramverk får användas.

 ## Krav
 * Webbplatsen ska använda minst två stycken olika webbtjänster/API:er, och dessa skall på något sätt kommunicera med varandra,
 * JavaScript-koden skall vara dokumenterad med JSDoc, 
 * Du ska använda en automatiserad arbetsprocess för att sammanställa och optimera webbplatsen samt sköta automatisk publicering. Denna arbetsprocess skall använda Git som grund för versionshantering och publicering,
 * Du ska använda ett versionshanteringssystem (Git) för att hantera källkod och dokumentation,
 * Du ska använda SASS/SCSS för styling, i motsvarande omfattning till tidigare laboration,
 * Du ska göra animeringar med hjälp av CSS och/eller JavaScript. Animeringar ska ha ett tydligt syfte som dokumenteras i rapporten,
 * Webbplatsen skall ha en välgjord design och grafisk profil, som är anpassad för den valda målgruppen,
 * Webbplatsen fungerar på olika typer av enheter med responsiv webbdesign,
 * Merparten av webbplatsens användargränssnitt och innehåll ska skapas och uppdateras med JavaScript och AJAX,
 * Webbplatsens innehåll, inklusive grafiskt material, skall vara optimerat.
